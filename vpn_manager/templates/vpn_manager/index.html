    Simple VPN

    {% trans "This module lets you manage your VPN connections." %}

    {% trans "Quick Reference" %}
    {% trans "Creating a peer" %}
    1. {% blocktrans %}Navigate to the VPN panel, then click on New.{% endblocktrans %}
    2. {% blocktrans %}Fill in the form with a name for this peer (for example, "My Smartphone"){% endblocktrans %}
    3. {% blocktrans %}Optional for an extra layer of security, provide your own public key. Leave it empty for auto-generation{% endblocktrans %}
    4. {% blocktrans %}Select Redirect traffic to forward all traffic through the VPN (standard usage){% endblocktrans %}
    5. {% blocktrans %}Select Killswitch if you want any traffic not going through the VPN to be dropped. Uncheck this option to continue accessing your local network (eg. printers, smart devices and such) even when connected to the VPN{% endblocktrans %}
    6. {% blocktrans %}Click on Save{% endblocktrans %}
    {% trans "Downloading configurations" %}
    1. {% blocktrans %}Select the VPN panel{% endblocktrans %}
    2. {% blocktrans %}From the list, click on the Options button matching the peer you want to see the configuration of, then select Show Configuration from the menu{% endblocktrans %}
    3. {% blocktrans %}Incomplete configuration if the private key is missing, a warning will be shown. In that case, copy the configuration and replace the private key field manually. You may use the QRCode Tool to generate a convenient valid QR Code{% endblocktrans %}
    4. {% blocktrans %}Download the configuration or scan the QR code{% endblocktrans %}
    Sharing configurations
    1. {% blocktrans %}Select the VPN panel{% endblocktrans %}
    2. {% blocktrans %}From the list, if the peer has a Shared badge, click on it. If it doesn't, click on the Options button then select Share from the menu. You will be asked to create a new link: press on the Create link button to do so. The link will then appear, alongside the Shared badge. {% endblocktrans %}
    3. {% blocktrans %}Copy the link provided and share it.{% endblocktrans %}
    4. {% blocktrans %}Disabling the link you may disable the link anytime.{% endblocktrans %}
